## 🚀 Featured Production Projects

### 1. 🏥 MediCare — Doctor Appointment Platform
A secure, production-grade health-tech workspace designed to connect patients with clinical services smoothly.
* **Core Mechanics:** Dynamic doctor discovery dashboards, real-time availability slots, and custom administrative controls.
* **Security Layer:** State-managed backend route protections integrated with modern **Better Auth** frameworks and Google OAuth sign-ins.
* **Tech Stack:** Next.js, React.js, Tailwind CSS, DaisyUI, Node.js, Express.js, MongoDB, Vercel.
* 🔗 [Live Demo](https://assignment-09-kappa.vercel.app/home)

### 2. 🎓 SkillSphere — Modern E-Learning Platform
A sleek, high-performance web platform built for technical professional development and digital classrooms.
* **Core Mechanics:** Nested accordion syllabus menus, trending catalogs, active global course searches, and live profile updates.
* **UI/UX Experience:** Immersive cyber-inspired UI animations driven by **Framer Motion** transitions alongside instantaneous form-validations.
* **Tech Stack:** React.js, Tailwind CSS v4, DaisyUI, Better Auth Integration, Vercel.
* 🔗 [Live Demo](https://assignment-008-wine.vercel.app/)

### 3. 🛠️ DigiTools Platform — Digital Product Manager
An efficient workspace application developed to optimize interactive digital resource organization and product discovery.
* **Core Mechanics:** Prevention algorithms countering product card duplication, continuous real-time state basket modifications, and responsive view ports.
* **Tech Stack:** JavaScript (ES6+), React, Vite, Tailwind CSS, DaisyUI, React Toastify, Netlify.
